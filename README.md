# Siddhant Rao — Portfolio

Interactive portfolio and resume website for job applications.

## Features

- Clean, professional design with dark/light mode
- Smooth scroll navigation and scroll-triggered animations
- Interactive skill chips and impact metrics
- Responsive layout for mobile and desktop
- One-click contact actions (email, LinkedIn, copy email)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Resume PDF

Place your resume PDF at `public/resume.pdf` for the download button to work.

## Deploy

Build for production:

```bash
npm run build
npm run preview
```

Deploy the `dist` folder to Vercel, Netlify, or GitHub Pages.

## Customize

Edit content in `src/data/profile.ts` to update your information.
